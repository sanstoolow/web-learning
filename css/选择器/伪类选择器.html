  • 类选择器无法实现这种效果
    .no { color : orange; } <a class="no" href="www.baidu.com">去百度</a> 因为这样无法选择点击前和点击后
  • <style>
    a:link {
    color: orangered; }
    a:visited {
    color: darkblue; }
    ;/style>
  • 伪类：很像类但是并不是，带上状态的类而已
  • 举例：
    <style>
    a:link 未访问{
    color: orangered; }
    a:visited 访问过{
    color: darkblue; }
    ;/style>
    a:hover 鼠标悬浮时{
    color: darkblue; }
    ;/style>
    a:active 点击未松手{
    color: darkblue; }
    ;/style>
    1. link
    2. link hover
    3. link hover active
    4. access
  • 因为点击a标签经过以上四个过程，每一个都是重合的，css的顺序不能变！！

结构伪类_2

  • 在1中介绍了什么为first-child
  • 相似的我们也有许多变种
<style>
div>p:last-child {
color:red;
}
</style>

即指div所有的子元素的最后一个是否是p，是则染色(按所有元素来看)

如果我想看特定顺序的child怎么办


<style>
div>p:nth-child(3) {
color:red;
}
</style>

指明的第3个元素是否是p，如果是则染色

括号内可以填n 2n 2n+1 even(偶数) odd(奇数)一类，自行看look

n值从0开始增加且必须写成an+b的形式，渲染前五个是写作-n+5

<style>
div>p:first-of-type {
color:red;
}
</style> <style>

}
</style>

在同元素的兄弟之间第一个或者最后一个！
