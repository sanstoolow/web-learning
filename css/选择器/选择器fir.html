选择器

基本选择器有以下几种

  1. 通配选择器
  2. 元素选择器
  3. 类选择器
  4. id选择器

基础选择器

通配选择器

  • 作用：选中所有的html元素
  • 语法：
    * { color : blue; }
  • 举例:
    * { color: orange; font-size: 40px; }
  • 点击此处查看全局效果

貌似没有用处，但是可以用于清除样式

元素选择器

  • 作用：选中特定的html元素
  • 语法：
    h2 { color: orange; font-style: italic; }

元素选择器无法形成差异化，通过类选择器可以实现差异

类选择器

  • 作用：通过class 类型来渲染某些元素
  • 举例：
    .answer { color : blue; } .big { font-style: italic; }
  • 如果使用两个类型需要如后使用class="big answer"

元素的class属性不带点，但是类选择中需要带


class值是由认为定义的，按照标准不允许使用纯数字，中文，要做到见名知意


多个class值已经如上演说过不再过多解释

id选择器

  • 作用：给特定的id的元素加上style
  • 举例：
    #earthy { color: red; }
  • id是独一无二的元素标记，每个元素的id不同

id不可以以数字开头，css严格要求


id属性和class可以同时有

其余选择器

交集选择器

  • 作用：多个条件限制下交集条件形成的特定元素（起且的作用）
  • 举例：
    p.beauty#wc { color: blue; }
  • 交集条件可以是多种多样的，但是不可以是错误的 如p.span两个元素限制明显不对
  • 举例：
    .beauty.clll { font-style: italic; }
  1. 有标签名，标签名必须写在前面
  2. id选择器，理论上可以作为选择条件实际上用的很少
  3. 用的最多的p.beauty这样的选择器

并集选择器

  • 作用：符合多个条件中的一个的元素渲染（起或的作用）
  • 举例：
    .rich { color : blue; font-size : 60px; backgroundcolor: red; width: 1000px } .ff { color : blue; } .dd { font-size: 60px; } .ddd { width: 200px; } /* 并在一起*/ .ff, .dd, .ddd{ color : blue; font-size : 60px; width: 1000px } #kun, .dingzhen, #kunkun { font-style: italic; }
  • 并集选择器一般竖着写，而且并不是类选择器可以用，id选择器也可以用
  • 不同选择器之间需要逗号

后代选择器

  • 作用：选择某一个元素的后代的进行渲染
  • 举例：
    ul li { color: red; } /*意思即为所有ul的后代（不一定是儿子可以是孙子）li都是渲染为红色*/ ol li { color: green; } ul li a { color: orange; } /*意思即为所有ul的后代li后代的a都是渲染为橙色*/ .subject li{ color : mind; } .subject #chik{ font-style : italic; } /*这个选择器很简单，可以用类，元素或者id都可以*/
  • 后代选择器是落在元素上的
  • 结构要符合html的要求,不能写错

子代选择器

  • 作用：更加精细化的渲染子代中特定元素而不是一棍子打死
  • 举例：
    div>a { color: red; } /*div元素的子元素（不是后代！）a渲染为红色*/ div>p>a { color: mind; } /*div元素的子元素p的子元素a渲染为红色*/ div>p>div>a { font-size: 60px; } /*但是上面这个不起作用，因为可能出现重复的结构导致冲突*/ #foot>a { font-style : italic; } .cav>p>a { font-style : }

特别提醒：子代和后代需要分清楚！！！

兄弟选择器

  • 作用：渲染兄弟元素
  • 举例:
    div+p { color: red; } /*div和p紧紧相邻的兄弟元素p颜色都渲染为红色，一般只看下面即div下面的p*/ /*睡在下铺的兄弟*/ div~p { color: mind; } /*div所有的兄弟p元素为渲染为雾色*/ /*和+类似也是从div下面的开始数*/ li+li{ color : orange; }
  • 选中的是兄弟而不是自己！！！！

相邻用+ 通用的为~ 均选择下面的兄弟

属性选择器

  • 作用：对有特定属性的元素渲染
  • 举例：
    第一种： [title] { color: red; } 具有title属性的元素

    第二种: [title="chicken"]{ color: mind; } 具有值为chicken的title的元素
    第三种： [title^="x"] { color: orange; } 具有以x开头的title值的元素
    第四种： [title$="x"] { color: orange; } 具有以x结尾的title值的元素
    第五种： [title*="x"] { color: orange; } 具有x字母的title值的元素
  • 属性选择器五种，一般不写class因为可以直接.school，一般用title这种填如

选择器优先级

不同的选择器，选择相同的元素，设置不同的样式，对于每一种样式，有以下的优先级。

  • 元素选择器>通配选择器
  • 类选择器>元素选择器
  • id选择器>类选择器
  • 行内样式>id选择器
